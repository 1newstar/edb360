  10. ENH: SQL Baselines and SQL Profiles are now summarized by type and by creation month.
  11. ENH: Active Session History (ASH) includes now "Top PLSQL Procedures" section.
  12. ENH: Tables with more than 5 Indexes are now reported under Database Administration.
  13. ENH: Added a query on GV$SYSTEM_WAIT_CLASS under Performance Summaries.
  14. ENH: Added a query on GV$OSSTAT under Operating System (OS) Statistics History.
